

# ALL ABOUT LILIKO (关于力力口的一切)

> **A dimension-flipping puzzle platformer born in 48 hours.**

**ALL ABOUT LILIKO** follows the journey of a cat named Liliko who accidentally falls into a computer. To return to the real world, you must manipulate a **Mask mechanism** to reveal hidden paths between two overlapping digital dimensions.

## Core Mechanics: The Mask

The game centers on the interaction between two parallel worlds. By toggling the **Mask**, players can:

* **Shift Reality:** Change terrain, platforms, and hazards in real-time.
* **Logic Puzzles:** Use the "inside" and "outside" of the mask to bypass obstacles that seem impossible in a single dimension.

## Key Features

* **Innovative Gameplay:** Unique dual-world switching logic that redefines 2D platforming.
* **Stunning Aesthetics:** High-fidelity art style blending digital glitch themes with charming character design.
* **Refined Controls:** Polished physics and responsive movement for a fluid "game feel."
* **Intelligent Design:** Clever puzzles that challenge both your logic and your reflexes.

## Project Origin

Originally developed for **Global Game Jam 2026** within a strict **48-hour** timeframe.

* **Award:** Winner of the *Most Popular Game Award*.
* **Status:** Prototype / Proof of Concept.

## Controls

* **Move:** `A/D`
* **Sprint:** `Shift`
* **Jump:** `Space`
* **Move Mask:** `Mouse Click & Drag`
* **Recall Mask:** `R`
* **Resize Mask:** `Scroll Wheel`
